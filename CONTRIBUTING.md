# Contributing to myapi.rest (Hub)

This repository is the central hub for the **myapi.rest** API suite. It contains:

- High-level documentation
- Cross-API examples
- Links to individual API repos

We welcome:

- Documentation improvements
- Corrections & clarifications
- Suggestions for new examples or sections

Please:

- Use GitHub **Issues** for bugs and suggestions.
- Use **Discussions** for open-ended questions and ideas.

By participating, you agree to follow our [Code of Conduct](./CODE_OF_CONDUCT.md).
